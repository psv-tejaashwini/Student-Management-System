# Student Management System

A comprehensive desktop-based Student Management System built using Java, featuring student record management, course enrollment, and grade tracking with MySQL database integration.

## Features

- **Student Management**: Add, update, delete, and view student records
- **Course Management**: Manage courses with detailed information
- **Enrollment Tracking**: Handle student enrollment in courses
- **Grade Management**: Track and update student grades
- **Database Integration**: MySQL database connectivity with JDBC
- **GUI Interface**: Intuitive desktop interface using Swing/AWT

## Technologies Used

- Java (JDK 11+)
- MySQL Database
- JDBC for database connectivity
- Swing/AWT for GUI development
- Maven for project management

## Prerequisites

- Java Development Kit (JDK 11 or higher)
- MySQL Server
- Maven (optional, if using Maven build)
